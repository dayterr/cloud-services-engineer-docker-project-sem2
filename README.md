# Общая информация
Приложение состоит из 2 компонентов:
- backend – написан на Go, ради безопасности используется образ Go 1.22, слушает порт 8081
- frontend – написан на Vue.js, ради безопасности используется образ Node 20, слушает порт 80

# Deploy 
Присутствует шаг проверки на уязвимости – Trivy и Snyk (Docker Scout), что способствует глубокой проверке на наличие уязвимостей.

# Docker Compose
Есть 2 файла:
- dev-версия – docker-compose.dev.yml, предназначен для тестового использования
- prod-версия – docker-compose.yml
Оба файла потребуют переменную `JWT_TOKEN`, prod-версия также требует `DOCKER_USERNAME`.

# Ограничения и возможности
